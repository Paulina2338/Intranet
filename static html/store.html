Safe File Transfering with Filezilla

Steps for File Transfer Process Step 1

Open Filezilla and add your provided login credentials. (This information will be provided by your manager).

Step 2

This is your file homescreen. The folder to your left is the host folder, here you will be able to add, take and consult company documents.

Step 3

To transfer documents between folders select files to transfer, give right clic and select Add files to queue.

Step 4

Select Process Queue to submit the request for file transfer. When File is transferred succesfully a pop up message will appear.
